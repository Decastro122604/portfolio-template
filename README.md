## ✨ Features

| Feature | Description |
|---|---|
| 🎨 **Claymorphism + Glassmorphism** | Unique combo of puffy clay buttons and frosted glass cards |
| 🌿 **Water Lily Palette** | Dark Green, Moss Green, Beige, Rosy Brown, Midnight Green |
| 🌸 **Custom Flower Cursor** | Animated 5-petal flower cursor that reacts on hover & click |
| 🌙 **Dark / Light Mode** | Smooth theme toggle with icon switch |
| ⌨️ **Typing Animation** | Hero role cycles through your custom phrases |
| 📊 **Animated Skill Bars** | Progress bars with category filter (All / Frontend / Backend / Tools) |
| 🔢 **Number Counter** | Stats animate from 0 to your values on scroll |
| 🎊 **Confetti Burst** | Confetti explodes on badge hover |
| 🌺 **Floating Flowers** | Tiny animated flowers float across the background |
| 📜 **Scroll Progress Ring** | Back-to-top button with circular progress indicator |
| 🃏 **3D Tilt Cards** | Project cards tilt following your mouse |
| 🔔 **Copy Email Toast** | Click to copy email with animated toast notification |
| 📱 **Mobile Responsive** | Hamburger menu + mobile-friendly layout |
| 🖨️ **Print-to-PDF Ready** | Polished CV output when printed |
| 🖼️ **Page Loader** | Initials animation before content appears |
| ✨ **Fade-in Animations** | Elements animate in as you scroll |

## 🚀 How to Use

### Step 1 — Open the file
Open `portfolio-template.html` in **Chrome** or **Edge** (for best experience).

### Step 2 — Fill up the form
Fill in all your details:
- 📸 Upload your profile photo
- 👤 Personal info (name, role, email, LinkedIn, GitHub)
- 📊 Stats (years of experience, tech count, projects)
- ⌨️ Typing animation phrases
- 🛠️ Skills with category and percentage
- 💼 Projects, Experience, Education, Certifications

### Step 3 — Generate!
Click **"Generate My Portfolio!"** and watch your portfolio come to life! 🌸

### Step 4 — Save as PDF (optional)
Press `Ctrl+P` → Save as PDF → your polished CV is ready!
